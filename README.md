# SIPNA - Sistema Acadêmico Java

Projeto de refatoração de um sistema acadêmico desenvolvido em Java.

## Objetivo

Aplicar boas práticas de desenvolvimento como:

- Separação de camadas
- Princípios SOLID
- DAO Pattern
- Service Layer
- Organização de pacotes

## Estrutura do Projeto


- br.com.sipna
- model
- dao
- service
- database
- main


## Tecnologias utilizadas

- Java
- JDBC
- MySQL
- Programação orientada a objetos
